## Project Overview

This is a blood report analysis system that uses AI to interpret medical test results. The system can:

1. Extract text from PDF and image files using OCR
2. Parse blood test parameters and values
3. Provide AI-powered analysis and recommendations
4. Compare multiple reports for trend analysis
5. Offer interactive chat for questions about results

## Architecture

The system is built with:
- **Frontend**: Streamlit web interface
- **Backend**: Python with multiple processing modules
- **AI**: Ollama with Mistral model for intelligent analysis
- **OCR**: Tesseract for text extraction from images/PDFs
